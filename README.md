# TraceMap

TraceMap is a user behavior analytics platform that helps analyze how users interact with web applications through tracking, replay, visualization, and behavioral insights.

## Features

### Session Tracking

* Unique session identification
* Session management and creation
* Session-based event collection

### Session Replay

* Cursor movement replay
* Click replay
* Scroll replay
* Screenshot-based session playback
* Adjustable replay speed controls

### Heatmap Analytics

* Click Heatmaps
* Mouse Movement Heatmaps
* Scroll Heatmaps
* Dead Click Heatmaps
* Rage Click Visualization

### User Behavior Tracking

* Click Tracking
* Mouse Movement Tracking
* Scroll Tracking
* Dead Click Detection
* Rage Click Detection

### Analytics Dashboard

* Session Analytics
* Page Analytics
* Interaction Metrics
* User Engagement Metrics
* Scroll Depth Analytics

### Behavioral Insights

* Frustration Detection
* Engagement Analysis
* Deep Content Exploration Detection
* Quick Disengagement Detection
* Session-Level Reasoning Layer
* Rule-Based Behavioral Insights

### Screenshot Capture

* Automatic session screenshots
* Manual screenshot capture support
* Replay screenshot visualization
